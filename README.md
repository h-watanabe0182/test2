# ファイル検索ツール

指定したディレクトリ以下のファイルから文字列を検索するシンプルなコマンドラインツールです。

## 使い方

```
python search_tool.py DIRECTORY QUERY [--ignore-case] [--ext EXT]
python search_tool.py --gui
```

- `DIRECTORY` ... 検索したいフォルダのパス
- `QUERY` ... 探したい文字列
- `--ignore-case`, `-i` ... 大文字小文字を区別しない
- `--ext` ... 指定した拡張子のファイルのみ検索する（例: `.txt`）
- `--gui` ... GUI で検索を行う

GUI を利用するには `DISPLAY` 環境変数が設定されている必要があります。環境によって
は GUI を起動できない場合があります。

引数を省略して実行すると、自動的に GUI が起動します。

一致した行は `ファイルパス:行番号:行の内容` の形式で表示されます。
